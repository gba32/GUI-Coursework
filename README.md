# GPX Weather App

Overview:

This project is a web-based application designed to provide weather forecasts along a user-defined route, specifically tailored for athletes such as cyclists, runners, and outdoor enthusiasts.
Users can upload a GPX file, and the app generates a weather forecast across the entire route, helping them plan activities more effectively and safely.
We believe athletes would greatly benefit from this app as it would allow for detailed location-based forecasting avoiding any weather they aren't prepared for.


Features:
- Upload GPX files
- Visualize routes on an interactive map
- Weather forecast along the entire route
- Time-based weather estimation (depending on route progression)
- Standard weather view for general use


How It Works:
1) User uploads a .gpx file
2) The app parses route coordinates
3) Weather data is fetched for points along the route
4) Forecast is displayed visually and/or along the path


Limitations:
- Accuracy depends on weather API data
- Forecasts may vary for long-duration routes
- Requires a valid GPX file
